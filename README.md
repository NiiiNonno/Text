# Text

## Bunan Code

### 符号表
坤|艮|坎|巽|震|離|兌|乾|
-|-|-|-|-|-|-|-
無|水|金|土|火|木|月|日
〇|一|二|三|四|五|六|七
八|九|癸|壬|辛|庚|己|戊
丁|丙|乙|甲|地|人|天|難

一段とニ段、三段と亖段は似ており、区別しなくてもよい。則ち、最上位ビットは無視して良い。

代表して、
`〇一二三四五六七八九癸壬辛庚己戊`
の十六字を無難数と呼ぶ。
